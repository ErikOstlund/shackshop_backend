## System Requirements

- [NodeJS][https://nodejs.org] `14 || 15 || 16`
- [npm][https://www.npmjs.com/] v6 or greater

These must be available in the project `PATH`. To verify, run this:

```shell
node -v  or  node --version
npm -v  or  npm --version
```

## Setup

```shell
npm install
```

## Running the app

```shell
npm run start
```

Go to: http://localhost:9000/
